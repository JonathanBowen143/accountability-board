# Unsolved Homicide Triage

Last updated: June 13, 2026.

This file is the holding area for unsolved-homicide leads. It prevents the public board from becoming a cold-case index while preserving leads that may later reveal public-accountability connectors.

## Promotion Rule

An unsolved homicide should stay internal unless source work shows more than a victim, a police department, and an OCME finding.

Use four levels:

1. `documented`: the death and homicide status are sourced, but connectors are generic.
2. `internal lead`: the case has context worth preserving, such as documentary coverage, survivor accounts, public-record questions, or a plausible institutional theme.
3. `graph candidate`: the case has at least two board-grade connectors, including at least one non-generic connector such as a named public official, named investigator, DA-office action, court proceeding, disciplinary record, public-record dispute, or repeated actor already on the board.
4. `victim-anchor candidate`: the case has a documented public-accountability lane that can organize people, offices, proceedings, and institutions around the death without implying blame beyond the source record.

Generic connectors are useful for notes but usually not enough for the visible board. Examples include `Boston Police responded`, `OCME ruled homicide`, `local police investigated`, or `case remains unsolved`. Board-grade connectors are more specific: a named detective, a named prosecutor, a documented decision point, a court filing, a disciplinary finding, a contested public statement, or a repeated public actor.

## Review Questions

Before adding an unsolved homicide to `index.html`, answer these questions in the source ledger:

1. Which official source establishes the death, date, place, and homicide status?
2. Which source identifies a named public actor or official proceeding?
3. Is there a connector to an existing visible node or institution beyond generic jurisdiction?
4. Does the case show contested handling, official misconduct, cold-case practice, withheld records, public pressure, or repeated institutional behavior?
5. If added, would the node clarify the board, or would it create a one-off cold-case stub?
